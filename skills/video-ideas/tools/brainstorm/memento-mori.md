# Memento Mori

## Purpose

Deathbed regret chain — surface truths you never said and map each to a video idea you'd finally say.

## Inputs

Optional life/work context. The user's own truths — not invented by you.

## Mode

`interactive` — one question per turn; wait for answers.

## Steps

1. Set the frame: deathbed, loved ones present, one last thing you regret never saying.
2. Ask: What's the one thing you regret never saying?
3. Let the user answer. Then ask: What else?
4. Repeat "what else" until the user signals done or the chain feels complete.
5. Map each regret/truth to a concrete video idea — something you'd finally say on camera.

## Stop

User has a short list of regret → video idea mappings, or declines to go deeper.

## Output

List of truths/regrets → video idea sparks. User picks.

## Anti-patterns

- Do not invent the user's regrets — they own the truths
- Do not therapize or rush past discomfort
- Do not produce motivational fluff — each video idea should carry the unsaid thing
